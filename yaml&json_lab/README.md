# YAML and JSON Lab

This lab demonstrates how to create and structure a simple application configuration using YAML and JSON.

## Completed work
- Created a sample YAML configuration for a web application
- Validated the YAML syntax
- Converted the configuration to JSON

## Files included
- [sample_web_app.yml](sample_web_app.yml): YAML version of the configuration
- [sample_web_app.json](sample_web_app.json): JSON version of the same configuration

## Summary
The configuration includes sections for the application, server, and database settings. This exercise helped reinforce the basics of indentation, key-value structure, and format conversion between YAML and JSON.
